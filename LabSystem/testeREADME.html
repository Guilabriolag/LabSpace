Bem-vindo ao LabSystem Store

Solução digital para lojas e restaurantes

🧭 Visão Geral

Popular: Sistema que ajuda sua loja a funcionar melhor, direto do navegador.

Moderada: Solução serverless que conecta gestão interna com atendimento ao cliente.

Técnica: Arquitetura modular com CMS e Totem, sincronização via JSONBin REST API.

✨ Funcionalidades

Popular: Cadastre produtos, personalize e receba pedidos pelo WhatsApp.

Moderada: Controle total do cardápio, visual e dados da loja.

Técnica: CRUD com localStorage, sincronização via PUT/POST, checkout com taxa dinâmica.

🛠️ Estrutura

Popular: Tem uma parte para o dono e outra para os clientes.

Moderada: Pastas separadas para CMS e Totem.

⚙️ Configuração

Popular: Crie conta no site, copie os códigos e cole no sistema.

Moderada: Configure BIN ID e Master Key no CMS. Totem pode ler ou ser configurado manualmente.

Técnica: JSONBin.io fornece credenciais. CMS publica via PUT, Totem consome via GET.

📝 Guia de Uso

Popular: Cadastre, personalize e publique. O cliente escolhe e finaliza no WhatsApp.

Moderada: CMS configura e sincroniza. Totem exibe e calcula entrega.

Técnica: CMS salva localmente, publica via JSONBin. Totem gera mensagem de checkout.

💻 Tecnologias

Popular: Usa HTML, cores bonitas e um site que guarda os dados.

Moderada: Front-end com Tailwind e JavaScript moderno. Backend via JSONBin.

Técnica: HTML5, Tailwind CSS, ES6+, JSONBin REST API, YouTube Iframe Player.
