LabSystem Store

CMS e Totem Serverless

Visão Geral do Projeto

O LabSystem Store é uma solução completa, leve e 100% serverless para o gerenciamento de uma loja de alimentos ou varejo. Ele consiste em dois módulos principais: um CMS para controle de cardápio e configurações, e um Totem de Pedidos para exibição pública e finalização via WhatsApp.

✨ Funcionalidades Principais

  • CMS: Backend Serverless, Persistência Local, CRUD, Backup/Restore, Customização Visual
  • Totem: Cardápio Dinâmico, Sistema de Carrinho, Entrega Inteligente, Checkout via WhatsApp

⚙️ Configuração Inicial

  1. Crie conta no JSONBin.io e obtenha BIN ID e X-Master-Key
  2. Configure o CMS com esses dados
  3. Configure o Totem com BIN_ID manualmente se necessário

📝 Guia de Uso

CMS: Preencha dados, salve rascunho, publique.

Totem: Cliente navega, adiciona ao carrinho, finaliza via WhatsApp.

💻 Tecnologias Envolvidas

  • Tailwind CSS
  • JSONBin.io
  • API do YouTube Iframe Player
